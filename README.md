# 简介📃

`vue-kun-admin` 是一个后台管理系统模板，采用 `Vue`、`TypeScript` 等最新前端技术栈进行开发，该系统仅用作学习参考📚

## 克隆仓库📦

```
# 从Github上拉取
git clone https://github.com/kun-js/vue-kun-admin.git
```

## 安装依赖

```
cd vue-kun-admin
pnpm install
```

## 项目启动

```
pnpm dev
```

## 项目打包

```
pnpm build
```
