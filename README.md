# 简介 📃

`vue-kun-admin` 是一个中后台管理系统模板，采用 `Vue`、`TypeScript`、`Element-Plus` 等最新主流前端技术进行开发，该系统仅用作个人学习参考 📚

## 项目相关 🧭

- 项目仓库地址：https://github.com/kun-js/vue-kun-admin

## 克隆仓库 🏠

从 Github 上拉取

```
git clone https://github.com/kun-js/vue-kun-admin.git
```

## 安装依赖 🏀

```
cd vue-kun-admin
pnpm install
```

## 项目启动 🚀

```
pnpm dev
```
